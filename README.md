# Java Jump – Course Projects & Exercises (2025)

This repository contains the code, exercises, and projects I’m building as part of the **Java Jump course** (July–September 2025). The course provides hands-on training in core Java, OOP principles, databases, testing, and backend development using Spring Boot.

## 📚 Course Scope

✅ Java SE fundamentals (variables, loops, arrays, regex, exceptions) \
✅ Object-Oriented Programming (classes, inheritance, polymorphism, SOLID) \
✅ Testing with JUnit 5, refactoring & clean code \
✅ File I/O and basic concurrency \
✅ SQL & relational databases \
✅ ORM with JPA/Hibernate, PostgreSQL and H2 \
✅ REST API development with Spring Boot & Spring Data JPA \
✅ Final project – complete backend application \

## 🛠️ Technologies Used

- Java 17
- IntelliJ IDEA (Community Edition)
- Git & GitHub
- PostgreSQL / H2 Database
- JUnit 5
- Spring Boot 3.x
- Maven

This repository documents my learning journey and serves as a showcase of my skills as a junior Java developer. All code is written and committed weekly throughout the course.
